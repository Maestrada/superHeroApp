# superHeroApp
Aplicacion para consumir json de una api

Cambiar el token del API en el Web.Config Key <heroToken>

Si da algun problema en la ejecucion del proyecto como ejemplo «could not find file…\bin\roslyn\csc.exe»
Para resolver este error basta con actualizar con el siguiente comando: PM > update-package Microsoft.CodeDom.Providers.DotNetCompilerPlatform
y despues reiniciar el VS
